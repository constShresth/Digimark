About Digital Marking System

Overview

The Digital Marking System is an AI-powered solution designed for the District Development Office in Banaskantha to automate the evaluation of handwritten exam papers. This system reduces the time and effort required for manual evaluation, ensures consistency in marking, and provides detailed feedback and analytics.

Key Features

  • Automated Evaluation: Uses OCR technology to extract text from scanned answer sheets and evaluate them against model answers.
  • Keyword Analysis: Identifies key concepts in student answers and compares them with expected keywords.
  • Semantic Similarity: Measures how closely the student's answer aligns with the model answer in meaning.
  • Detailed Feedback: Provides specific feedback on what concepts are missing or well-addressed in each answer.
  • Performance Analytics: Generates visual representations of student performance across questions.
  • PDF Reports: Creates comprehensive PDF reports that can be downloaded and shared.

How It Works

  1. Upload a scanned answer sheet (JPG or PDF format)
  2. Upload an answer key (JPG, PDF, or TXT format)
  3. The system extracts text from the answer sheet using OCR
  4. Answers are evaluated against the model answers using keyword matching and semantic similarity
  5. Scores and feedback are generated for each answer
  6. Performance analytics and visualizations are created
  7. A comprehensive report is generated that can be viewed online or downloaded as PDF

Benefits

  • Time Savings: Reduces evaluation time by up to 70%
  • Consistency: Ensures uniform evaluation criteria across all papers
  • Detailed Insights: Provides analytics that help identify knowledge gaps
  • Reduced Workload: Frees up teachers to focus on teaching rather than grading
  • Faster Feedback: Students receive detailed feedback much sooner
